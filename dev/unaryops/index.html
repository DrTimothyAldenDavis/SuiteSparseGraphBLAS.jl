Unary Operators

UnaryOps are fairly straightforward $z = f(x)$ and their meaning should be clear from the name in most cases.

UnaryOps are used only in the map function, for example:

julia> using SuiteSparseGraphBLAS

julia> x = GBVector([1.5, 0, pi])
3x1 GraphBLAS double vector, full by col
  3 entries, memory: 280 bytes

    (1,1)    1.5
    (2,1)    0
    (3,1)    3.14159

julia> y = map(UnaryOps.SIN, x)
3x1 GraphBLAS double vector, full by col
  3 entries, memory: 288 bytes

    (1,1)    0.997495
    (2,1)    0
    (3,1)    1.22465e-16

julia> map(UnaryOps.ASIN, y)
3x1 GraphBLAS double vector, full by col
  3 entries, memory: 288 bytes

    (1,1)    1.5
    (2,1)    0
    (3,1)    1.22465e-16
