# SuiteSparseGraphBLAS.jl
A fast, general sparse linear algebra and graph computation package, based on SuiteSparse:GraphBLAS.

### Installation
```julia
using Pkg
Pkg.add("SuiteSparseGraphBLAS")
```
Note: This package supports Julia 1.6+
